# Bidule — Releases

Ce dépôt ne contient aucun code source. Il héberge uniquement les
installeurs publiés de [Bidule](https://github.com/Quentin-Red-Turtle/Bidule)
(dépôt privé), pour que l'application puisse vérifier et télécharger ses
propres mises à jour via `tauri-plugin-updater`, sans authentification.
